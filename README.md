# User Data Analytics Pipeline

## Overview
This project demonstrates a simple data pipeline using Python and SQL.

## What I Did
- Extracted data from a public API
- Transformed and cleaned the data using Pandas
- Loaded the data into a SQL database
- Performed analysis using SQL queries

## Tech Stack
- Python
- Pandas
- SQL (SQLite)
- Google Colab

## Key Learnings
- Handling API data (JSON format)
- Working with nested data (like extracting city from address)
- Writing SQL queries for analysis

## How to Run
Open the notebook and run all cells step by step.
